# self-hosted-media-stack
A fully containerized self-hosted media server running on a UGREEN NAS, featuring Plex, Sonarr, Radarr, Prowlarr, Flaresolverr, Overseerr, qBittorrent routed through Gluetun (AirVPN), and a Discord automation bot (Doplarr). Includes secure VPN networking, volume mappings, automated downloads, and a documented deployment guide.
